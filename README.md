# Game Songs

游戏背景音乐合集，收录多款经典游戏的 BGM（如植物大战僵尸、超级马里奥、魂斗罗、俄罗斯方块、极品飞车、跑跑卡丁车等）。音频按游戏/系列分目录存放。

## 如何获取更多音乐

可使用 [爱给网 (aigei.com)](https://aigei.com) 配合浏览器扩展 **Video DownloadHelper** 免费下载游戏音乐。
