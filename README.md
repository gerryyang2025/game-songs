# Game Songs

游戏背景音乐合集，收录多款经典游戏的 BGM（如植物大战僵尸、超级马里奥、魂斗罗、俄罗斯方块等）。

## 目录结构

```
game-songs/
├── README.md
├── LICENSE
└── music/           # 音频文件（MP3）
    ├── contra.mp3
    ├── tetris.mp3
    ├── game_plants_vs._zombies*.mp3
    ├── game_mario*.mp3
    └── ...
```

## 内容说明

- **contra** — 魂斗罗
- **tetris** — 俄罗斯方块
- **game_plants_vs._zombies** — 植物大战僵尸系列
- **game_mario** — 超级马里奥系列
- **game1–game17** — 其他游戏 BGM

## 如何获取更多音乐

可使用 [爱给网 (aigei.com)](https://aigei.com) 配合浏览器扩展 **Video DownloadHelper** 免费下载游戏音乐。
